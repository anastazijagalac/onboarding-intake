Connect to Google Sheets

Paste your Google Apps Script Web App URL below. Every new joiner registered will be automatically saved to your shared Google Sheet. Follow the setup guide below to get your URL.

Step-by-step setup guide
1
Create a new Google Sheet
Go to sheets.google.com → click Blank → name it Ataccama Onboarding. You'll need two tabs: rename Sheet1 to Joiners and create a second tab called Sessions.
2
Open Apps Script
In your Google Sheet, click Extensions → Apps Script. A new tab opens with a code editor.
3
Paste the script
Delete everything in the editor and paste this script:
function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var ss = SpreadsheetApp.getActiveSpreadsheet();
    var sheet = ss.getSheetByName('Joiners');
    if (!sheet) sheet = ss.insertSheet('Joiners');

    if (sheet.getLastRow() === 0) {
      sheet.appendRow([
        'Timestamp','Name','Email','Start Date',
        'Team','Cycle','Sessions','Notes'
      ]);
      sheet.getRange(1,1,1,8).setFontWeight('bold')
           .setBackground('#1A0C42').setFontColor('#ffffff');
    }

    sheet.appendRow([
      new Date().toLocaleString(),
      data.name, data.email, data.startDate,
      data.team, data.cycle,
      data.sessions.join(', '), data.notes
    ]);

    return ContentService
      .createTextOutput(JSON.stringify({status:'ok'}))
      .setMimeType(ContentService.MimeType.JSON);

  } catch(err) {
    return ContentService
      .createTextOutput(JSON.stringify({status:'error',message:err.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet(e) {
  return ContentService
    .createTextOutput(JSON.stringify({status:'ok',message:'Ataccama Onboarding API running'}))
    .setMimeType(ContentService.MimeType.JSON);
}
4
Deploy as Web App
Click Deploy → New deployment. Set type to Web app. Under Execute as choose Me. Under Who has access choose Anyone. Click Deploy, authorise when prompted, then copy the Web App URL.
5
Paste the URL above and test
Paste your Web App URL in the field at the top of this page and click Test connection. If it says ✓ Connected — you're done!
